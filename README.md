# Student Management System 

Программный комплекс на языке Java для управления реестром студентов. Проект демонстрирует применение архитектурных паттернов, соблюдение принципов SOLID и использование возможностей многопоточной обработки данных.

## Ключевые особенности

*   **Надежное создание объектов (Pattern Builder)**: Гарантирует валидность данных (проверка 6-значного номера зачетки, диапазона оценок 0.0–5.0 и формата email) на этапе сборки объекта.
*   **Высокопроизводительная коллекция**: Собственная динамическая структура `StudentCollection` с поддержкой автоматического расширения и **Parallel Stream API** для мгновенного выполнения агрегатных функций и поиска.
*   **Гибкая система сортировок (Strategy & Template Method)**:
    *   Классическая сортировка через `StudentComparator`.
    *   Специализированная фильтрующая сортировка (только чётные поля) через абстрактный шаблонный метод `EvenFieldSortStrategy`.
*   **Контроль качества**: Наличие демонстрационного модуля (`Main`) и системы тестов (`StudentTests`) для верификации всей логики проекта.

##  Технологический стек

*   **Язык**: Java 17+ (Lambda, Stream API, Parallel Streams).
*   **Паттерны**: Builder, Strategy, Template Method.
*   **Инструменты**: Git, Regex (регулярные выражения в классе Student), Модульное тестирование.

##  Структура проекта

1.  **`Main`** — Точка входа, демонстрация сценариев и интеграция компонентов.
2.  **`Student`** — Модель данных с внутренним Билдером и валидацией.
3.  **`StudentCollection`** — Контейнер данных с поддержкой многопоточности и Stream API.
4.  **`StudentTests`** — Набор проверочных тестов для всех модулей системы.
5.  **Иерархия стратегий** — Набор интерфейсов и классов (`SortStrategy`, `StudentComparator`, `EvenFieldSortStrategy` и др.), обеспечивающих гибкую логику сортировки по любым полям.
